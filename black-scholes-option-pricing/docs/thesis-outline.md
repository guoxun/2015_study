# 论文章节梳理

> 原论文：《Black-Scholes 模型实现与基于隐含波动率的套利策略算法实现》
> Algorithmic Implementation of the Black-Scholes Model and Arbitrage Strategies based on Implied Volatility
> 北京交通大学计算机学院本科毕业设计 (2015.06)

## 摘要要点

Black-Scholes 模型为各类衍生金融工具的合理定价奠定了基础，使衍生证券定价能由计算机程序实现。模型有两大数学基础：**Ito 积分**与 **Wiener 过程**，在此基础上推导出无红利股票衍生证券价格所满足的微分方程，结合边界条件得到解析解（即 BS 公式）。

本文以恒生指数 1986-12-31 至 2013-06-28 交易日收盘价为样本，使用 Eviews 进行时间序列分析，建立股票对数收益率的 GARCH(1,1) 模型估计参数获取波动率，从而实现期权定价算法。在此基础上用牛顿迭代法实现隐含波动率求解，并实现各套利策略算法。最终用 Qt 框架与 C++ 实现完整的期权定价程序。

**关键词**：Black-Scholes 模型；期权定价；Wiener 过程；Garch 模型；波动率

## 七大基本假设

1. 股价随机过程的变化函数符合指数 Brown 运动模型
2. 允许卖空衍生证券
3. 无交易费用和税收
4. 没有红利支付
5. 没有无风险套利机会
6. 证券交易为连续行为
7. 无风险利率在合约时间区间内为常数

## 关键公式

### BS 微分方程

```
∂f/∂t + r·S·∂f/∂S + (1/2)·σ²·S²·∂²f/∂S² = r·f
```

边界条件 `f(S, T) = max(S − K, 0)` (看涨) 时得到 BS 解析解。

### 波动率微笑

执行价格远离标的资产现货价格时，隐含波动率会逐渐变大。在价期权 (at the money) 的波动率低于价外 (OTM) 与价内 (ITM)，曲线呈中间低两边高的"微笑"形状。该现象说明 BS 假设的恒定波动率与真实市场存在偏差。
